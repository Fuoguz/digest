# Digest · 沉淀 

> **“知识不仅是碎片，更是时间的结晶。”**

`Digest` 是一款专为深度学习者设计的“知识内化”智能体工具。它通过 Agentic Workflow（智能体工作流）将碎片化的媒介信息转化为结构化的个人认知节点，并利用 D3.js 力导向图实现知识的视觉化编织。

---

## ✨ 核心特性

### 1. 多源输入与媒介清洗
- **双通道导入**：支持直接粘贴、网页 URL、Markdown 及页面选区。内置“抓取失败自动镜像”功能，确保内容摄入的极高成功率。
- **语义清洗**：自动剔除 Markdown 杂质（Frontmatter、标题标记等），保留纯粹的叙事结构。

### 2. Agentic 深度拆解
- **智能体逻辑**：不同于简单的 Prompt 总结，Digest 采用 **Agent Loop** 机制。在生成过程中，AI 会自主决定是否检索本地图谱，实现新旧知识的自动关联。
- **结构化输出**：强制提炼“核心论点、论据、认知框架、复习问题”，拒绝信息发散。

### 3. 质量品控与防御
- **内置检查板**：实时计算完整度、置信度及偏题风险，为每一次沉淀提供质量预警。
- **输入锁定**：分析后自动锁定原文，防止误触，保持心流。

### 4. 知识沉淀体系
- **SRS 间隔复习**：内置 1/3/7/14/30 天复习调度算法，支持“待复习”视图。
- **D3 知识图谱**：力导向图实时展示节点关系，支持拖拽、高亮及“时间机”回放。
- **多端分发**：支持一键导出电影感海报 (PNG) 及 Markdown 笔记。

### 5. 极致视觉美学
- **电影感转场**：基于 `cubic-bezier(0.65, 0, 0.35, 1)` 的转场曲线，提供沉浸式的使用仪式感。
- **白盒化 UI**：实时展示 Agent 思考过程（Thinking Process），将黑盒 AI 透明化。

---

## 🛠️ 技术栈
- **Frontend**: Vanilla JS, D3.js (Visualization), HTML5, CSS3 (OKLCH Color System)
- **AI Engine**: OpenAI/Claude API (Supports Agentic Tool-Calling)
- **Storage**: Browser LocalStorage (Zero Server Dependency)

---

## 🚀 快速开始
1. **部署**：直接将代码上传至 GitHub Pages 或 Vercel 即可运行。
2. **配置**：在页面设置中填入你的 API Key。
3. **输入**：粘贴你想内化的文章 URL 或文本。
4. **沉淀**：点击“开始内化”，见证碎片变为知识。

## 🖋️ 设计理念
作为 **网络与新媒体** 背景的开发者，我致力于探索“人机协作”在个人知识管理中的新形态。`Digest` 不仅仅是一个摘要工具，它是我对信息过载时代的视觉化反抗——通过技术手段，强迫自己停下来，去观察、去拆解、去连接。
